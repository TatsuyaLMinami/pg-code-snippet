# pg-code-snippet README

A vue snippet plugin for vscode.

## 使用方法

| 关键字             | 模版               |
| ------------------ | ------------------ |
| vue-sfc            | 选项式vue          |
| vue-composition    | 组合式vue          |
| gl-layout          | 左侧树右侧列表布局 |
| gl-drawer          | 抽屉               |
| gl-table           | 表格               |
| gl-form            | 表单               |
| gl-tree            | 树                 |
| gl-card            | 卡片               |
| gl-page            | 分页               |
| gl-upload          | 上传               |
| gl-dialog          | 弹窗               |
| gl-dynamicForm     | 动态表单           |
| gl-dynamicFormItem | 动态表单项         |
